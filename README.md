# 🚀 Deploy Laravel Project on InfinityFree (Free Hosting)

This guide walks you through deploying a Laravel project to [InfinityFree](https://infinityfree.net), a free web hosting platform. It’s ideal for showcasing your Laravel apps without spending a dime.

---

## 📦 Requirements

- Laravel project (tested on Laravel 9+)
- GitHub repository
- InfinityFree account
- FTP client (e.g., FileZilla)
- `.env` file configured for production
